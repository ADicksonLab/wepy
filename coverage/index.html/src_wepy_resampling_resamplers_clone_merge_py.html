121 def assign_clones(self, merge_groups, walker_clone_nums): 

122 """Convert two convenient data structures to a list of almost 

123 normalized resampling records. 

124 

125 The two data structures are merge_groups and walker_clone_nums 

126 and are convenient to make. 

127 

128 Each is a list with number of elements equal to the number of 

129 walkers that resampling will act on. 

130 

131 Each element of the merge_groups is a list-like of integers 

132 indicating the indices of the walkers that will be merged into 

133 this one (i.e. squashed). A non-empty collection indicates a 

134 KEEP_MERGE decision. 

135 

136 Each element of the walker_clone_nums is simply an integer 

137 specifying how many clones to make of this walker. 

138 

139 These data structures simply declare requirements on what the 

140 actual decision records must achieve. The actual placement of 

141 walkers in slots (indices) is unspecified and immaterial. 

142 

143 Parameters 

144 ---------- 

145 merge_groups : list of list of int 

146 The specification of which walkers will be squashed and merged. 

147 

148 walker_clone_nums : list of int 

149 The number of clones to make for each walker. 

150 

151 Returns 

152 ------- 

153 

154 walker_actions : list of dict of str: values 

155 List of resampling record like dictionaries. These are not 

156 completely normalized for consumption by reporters, since 

157 they don't have the right list-like wrappers. 

158 

159 """

169 # the index of a merge group determines the KEEP_MERGE walker

183 target_idxs=(walker_idx,))

187 target_idxs=(walker_idx,))
