1"""Abstract base class for Decision classes. 

2 

3See the NoDecision class and others in this module for examples. 

4 

5To create your own subclass of the Decision class you must customize 

6the following class constants: 

7 

8- ENUM 

9- FIELDS 

10- SHAPES 

11- DTYPE 

12- RECORD_FIELDS 

13 

14The 'ENUM' field should be a python 'Enum' class created by 

15subclassing and customizing 'Enum' in the normal pythonic way. The 

16elements of the 'Enum' are the actual decision choices, and their 

17numeric value is used for serialization. 

18 

19The 'FIELDS' constant is a specification of the number of fields that 

20a decision record will have. All decision records should contain the 

21'decision_id' field which is the choice of decision. This class 

22implements that and should be used as shown in the examples. 

23 

24In order that fields be serializable to different formats, we also 

25require that they be a numpy array datatype. 

26 

27To support this we require the data shapes and data types for each 

28field. Elements of SHAPES and DTYPES should be of a format 

29recognizable by the numpy array constructor. 

30 

31To this we allow the additional option of specifying SHAPES as the 

32python built-in name Ellipsis (AKA '...'). This will specify the shape 

33as a variable length 1-dimensional array. 

34 

35The RECORD_FIELDS is used as a way to specify fields which are 

36amenable to placement in simplified summary tables, i.e. simple 

37non-compound values. 

38 

39The only method that needs to be implemented in the Decision is 

40'action'. 

41 

42This function actually implements the algorithm for taking actions on 

43the decisions and instructions and is called from the resampler to 

44perform them on the collection of walkers. 

45 

46"""

308 @classmethod 

309 def action(cls, walkers, decisions): 

310 

311 # list for the modified walkers 

312 mod_walkers = [None for i in range(len(walkers))] 

313 # go through each decision and perform the decision 

314 # instructions 

315 for walker_idx, decision in enumerate(decisions): 

316 decision_value, instruction = decision 

317 if decision_value == cls.ENUM.NOTHING.value: 

318 

319 # check to make sure a walker doesn't already exist 

320 # where you are going to put it 

321 if mod_walkers[instruction[0]] is not None: 

322 raise ValueError( 

323 "Multiple walkers assigned to position {}".format(instruction[0])) 

324 

325 # put the walker in the position specified by the 

326 # instruction 

327 mod_walkers[instruction[0]] = walkers[walker_idx] 

328 

329 

330 return mod_walkers
